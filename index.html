Términos y Condiciones de Uso

1. Aceptación de Términos

Al acceder y utilizar la API de Pacific Network, usted acepta estar sujeto a estos términos y condiciones de uso. Si no está de acuerdo con alguno de estos términos, no debe utilizar nuestra API.

2. Uso de la API

El uso de nuestra API está sujeto a las siguientes condiciones:

  • Debe mantener la confidencialidad de sus credenciales de API
  • No debe exceder los límites de tasa establecidos (1000 peticiones/hora)
  • Debe usar la API solo para propósitos legales y autorizados
  • No debe intentar eludir las medidas de seguridad

3. Seguridad y Privacidad

Pacific Network se compromete a:

  • Proteger la información de las transacciones con encriptación SSL/TLS
  • No almacenar información sensible de tarjetas de crédito
  • Cumplir con las normativas PCI-DSS
  • Notificar cualquier brecha de seguridad en un plazo de 72 horas

4. Limitación de Responsabilidad

Pacific Network no será responsable por daños indirectos, incidentales, especiales o consecuentes que resulten del uso o la imposibilidad de uso de la API, incluyendo pero no limitado a:

  • Pérdida de ganancias o ingresos
  • Pérdida de datos
  • Interrupción del negocio

5. Disponibilidad del Servicio

Garantizamos un uptime del 99.9% mensual. En caso de mantenimiento programado, notificaremos con al menos 48 horas de anticipación. Los tiempos de inactividad planificados no se contabilizan en el SLA.

6. Modificaciones del Servicio

Pacific Network se reserva el derecho de modificar o discontinuar la API en cualquier momento, con previo aviso de 30 días para cambios significativos que afecten la compatibilidad.

7. Soporte Técnico
